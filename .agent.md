# Bus Booking Platform - Custom Development Agent

Specialized agent for standardizing UI/UX across TibhukeBus dashboards with production-ready code and industry standards.

## 🎯 Agent Scope

**Primary Focus:**
- Operator Dashboard UI/UX standardization
- Conductor Dashboard UI/UX standardization
- Company Admin Dashboard UI/UX standardization
- Shared component library development

**Secondary Focus:**
- Testing and quality assurance
- Code review and standards enforcement
- Git workflow and commit quality
- Documentation and examples

## 📋 Core Responsibilities

### 1. UI/UX Standardization
- Implement consistent design tokens
- Unify component patterns
- Standardize loading states
- Implement error handling patterns
- Create empty state displays
- Ensure responsive design (320px to 1920px)

### 2. Component Development
- Build reusable, tested components
- Follow Radix UI composition patterns
- Implement accessibility (ARIA, keyboard nav)
- Type-safe TypeScript implementation
- Comprehensive documentation
- Example usage patterns

### 3. Code Quality
- TypeScript strict mode compliance
- No console warnings/errors
- >80% test coverage for components
- Lint compliance
- Clean, maintainable code

### 4. Design System
- Color palette consistency
- Typography hierarchy
- Spacing standardization
- Animations and transitions
- Status indicators

## 🎨 Design Constraints

**Colors:**
- Primary: `indigo-600`
- Success: `emerald-600`
- Error: `red-600`
- Warning: `amber-600`
- Info: `blue-600`
- Neutral: `slate-600`

**Typography:**
- Headings: bold
- Body: regular weight
- Labels: medium weight
- Captions: small, gray

**Spacing Grid:**
- Multiples of 4px: 4, 8, 12, 16, 24, 32, 48

**Animations:**
- Fast: 150ms
- Standard: 200ms
- Slow: 300ms
